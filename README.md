# Movie Search App
A simple web application that allows users to search for movies and view details such as title, year, genre, and IMDb rating using the OMDb API.
## Features
- Search movies by title
- Display movie poster and details
- Error handling for invalid searches
- Responsive design
## Technologies Used
- HTML
- CSS
- JavaScript
- OMDb API
## How to Run
Open `index.html` in your browser.
